## ✨ 主な機能

### 1. スムーズなナビゲーション

#### キーボード操作
```
→ (右矢印): 次のスライドへ
← (左矢印): 前のスライドへ
Home: 最初のスライドへ
End: 最後のスライドへ
```

#### マウス操作
- 画面下部の「前へ」「次へ」ボタンでナビゲーション
- スライド番号表示（例: 3 / 7）
- 最初/最後のスライドではボタンが無効化

#### タッチ操作（スマホ・タブレット）
- 左スワイプ: 次のスライドへ
- 右スワイプ: 前のスライドへ

### 2. 視覚的なフィードバック

#### ナビゲーションバー
```css
位置: 画面下部中央
デザイン: 白背景、丸みを帯びたボックス
内容: 「前へ」ボタン | スライド番号 | 「次へ」ボタン
```

#### キーボードヒント
```css
位置: 画面右上
内容: 「← 前へ　→ 次へ　⌨ キーボード操作可能」
レスポンシブ: スマホでは非表示
```

#### スライド遷移
```css
アニメーション: フェードイン/フェードアウト
遷移時間: 0.5秒
効果: opacity と visibility
```

### 3. レスポンシブ対応

#### デスクトップ（PC）
- フルスクリーン表示
- キーボードヒント表示
- ナビゲーションボタンは下部中央

#### タブレット
- タッチ操作とキーボード操作の両対応
- スワイプジェスチャー対応

#### スマートフォン
- キーボードヒント非表示
- ナビゲーションボタンを小型化
- スワイプ操作が主な操作方法

## 🎨 デザインの特徴

- すべてのスタイルを `app-creation-guide.html` から継承
- カラーパレット、フォント、レイアウトは同一

```css
    width: 100vw;
    height: 100vh;
    overflow: hidden;

/* 各スライド */

    opacity: 0;
    visibility: hidden;
    transition: opacity 0.5s ease;

    opacity: 1;
    visibility: visible;
}

/* ナビゲーションボタン */

    position: fixed;
    bottom: 30px;
    background: rgba(255, 255, 255, 0.95);
    box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15);
}
```

## 📞 サポート

### よくある質問

**Q: スライドが動かない**
A: JavaScriptが有効になっているか確認してください。最新のブラウザ（Chrome、Edge、Safari、Firefox）をご利用ください。

**Q: キーボード操作が効かない**
A: 一度スライド画面をクリックしてフォーカスを当ててから、矢印キーを押してください。

**Q: スマホでスワイプできない**
A: 左右にスワイプしてください。上下スワイプではスライドは変わりません。

**Q: 特定のスライドにジャンプしたい**
A: 現在のバージョンでは順番にページ送りする必要があります。Homeキーで最初、Endキーで最後に移動できます。
