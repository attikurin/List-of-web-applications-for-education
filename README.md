# 教育現場向けWebアプリ一覧サイト

小中学校の教員・ICT支援員のための便利ツールを集めた一覧サイトです。

## 🎯 プロジェクト概要

このサイトは、教育現場で活躍する先生方やICT支援員の方々に向けて、16個の実用的なWebアプリケーションを紹介する一覧サイトです。

### ✨ サイトの特徴

- **明るくポップなデザイン**: 教育現場に馴染む親しみやすいデザイン
- **16個のアプリを紹介**: 学級運営から授業支援まで幅広いツール
- **各アプリのイラスト付き**: カラフルで視覚的に分かりやすい
- **2つのアクションボタン**: 
  - 「アプリを開く」ボタン - 外部リンクでアプリにアクセス
  - 「HTMLダウンロード」ボタン - 各アプリのHTMLファイルをダウンロード
- **完全レスポンシブ対応**: スマホ・タブレット・PCで快適に閲覧可能
- **安全性の強調**: ブラウザ完結、インストール不要、データは端末に保存

## 📦 含まれるアプリ一覧

### 1. 全自動・顔モザイクメーカー
プライバシー保護に特化した顔モザイク加工アプリ。AI検出で高精度、動画対応。

### 2. クイック席替えメーカー
30秒で座席表が完成。Excelコピペ対応、印刷最適化。

### 3. 教室用ビジュアルタイマー
円グラフで残り時間を可視化。複数タイマー同時管理、全画面モード対応。

### 4. クラスメモボード
クラス情報を一箇所に集約。予定・連絡事項・当番表を管理。

### 5. ランダムセレクター
ルーレットモードで抽選、グループ分けモードで2～8チーム自動編成。

### 6. FormMirror
動作分析・フォームチェックアプリ。お手本動画と比較、AI骨格検出。

### 7. 万能PNGコンバーター
PDF、Office、画像をPNG変換。完全クライアント処理で機密保護。

### 8. QR Generator
教材用QRコード一括生成。Excel一括入力、印刷最適化レイアウト。

### 9. 教室シミュレーター
教室レイアウト配置シミュレーション。ドラッグ&ドロップで自由配置。

### 10. クラス運営マスター
学級運営オールインワンツール。日直・給食班・係活動を一画面で管理。

### 11. 名札メーカー Pro
首かけ名札と三角席札を簡単作成。CSV一括登録、印刷最適化。

### 12. 時間割メーカー
週間時間割を簡単作成。4つのデザインテーマ、児童用/保護者用モード。

### 13. デジタル出席簿
月間カレンダー型出席管理。クリックで出席○・遅刻△・欠席×を入力。

### 14. スマートグループ分け
最適なチーム分けを自動生成。男女比均等化、役割自動割り当て。

### 15. 提出物リネーム屋さん
バラバラなファイル名を一括整理。名簿リストやルールで自動リネーム、ZIP保存対応。完全ローカル処理で個人情報も安心。

### 16. 集中スポットライト
授業・プレゼンで視点を一点集中。3つの形状（円・四角・スリット）で資料の特定部分を表示。背景画像対応、透過率調整可能。タッチ操作対応。

## 🚀 主な機能

### 各アプリカードの構成

- **アプリイメージ**: 明るくカラフルなイラスト（16:9比率）
- **タイトル**: アプリ名とカテゴリーアイコン
- **タグ**: 機能や特徴を示すカラフルなタグ
- **説明文**: アプリの詳細説明
- **アクションボタン**:
  - **アプリを開く**: 外部サイトへのリンク（新しいタブで開く）
  - **HTMLダウンロード**: 各アプリのHTMLファイルをダウンロード

### HTMLダウンロード機能

各アプリのHTMLダウンロードボタンをクリックすると：
1. 対象アプリのURLからHTMLを取得
2. ブラウザでダウンロード処理を実行
3. `{アプリ名}.html`という名前で保存

**注意**: CORS（Cross-Origin Resource Sharing）ポリシーにより、一部のアプリでは直接ダウンロードできない場合があります。その場合は「アプリを開く」ボタンからアクセスし、ブラウザの「ファイル」→「名前を付けて保存」機能をご利用ください。

## 📖 使い方

1. **サイトを開く**: `index.html` をブラウザで開きます
2. **アプリを探す**: スクロールして興味のあるアプリを見つけます
3. **アクション選択**:
   - **アプリを使う**: 「アプリを開く」ボタンをクリック
   - **HTMLを保存**: 「HTMLダウンロード」ボタンをクリック
