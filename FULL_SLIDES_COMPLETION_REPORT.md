## ✨ 主な機能

### 1. **目次ボタン**

## 🎯 技術仕様

### JavaScriptによる動的生成
```javascript
// スライドデータ配列
const slidesData = [
    { type: 'hero', ... },
    { type: 'toc', ... },
    { type: 'intro', ... },
    { type: 'basic', ... },
    { type: 'category', ... },
    { type: 'app', ... },
    // ... 全33スライド分
];

// スライド生成関数
function generateSlides() {
    slidesData.forEach((data, index) => {
        // 各タイプに応じてHTMLを生成
        switch(data.type) {
            case 'app':
                // アプリスライドの生成
                break;
            case 'category':
                // カテゴリスライドの生成
                break;
            // ...
        }
    });
}
```

### スライドタイプ
1. **hero**: タイトル画面
2. **toc**: 目次
3. **intro**: はじめに
4. **basic**: 全アプリ共通の基本設定
5. **category**: カテゴリー紹介
6. **app**: 個別アプリ詳細（30個）
7. **steps**: Gemini活用の3ステップ
8. **tips**: プロンプトのコツ
9. **summary**: まとめ
